# Breakout

** Breakout is a 2D brick breaker game made following the completion of the book "LearnOpenGL"  **

---

- Player controlls the paddle via A and D keys for movment to the left and right respectively.
- There are 6 powerups that the player can catch with the paddle. Powerups include: size increase, ball passthrough, increased speed, sticky ball, **chaos** and **confusion**
